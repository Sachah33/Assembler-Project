# Assembler-Project
# Multi-Pass Assembler for Custom x86-like Architecture

## Overview
This repository contains a multi-pass assembler engineered from scratch in C. The software processes custom assembly language source files, executes macro pre-processing, performs lexical and syntax analysis, constructs dynamic symbol tables, and generates machine binary outputs.

The implementation simulates low-level CPU processing methodologies, including custom instruction registrations, addressing modes (Immediate, Direct, Relative, and Register), and data image handling.
